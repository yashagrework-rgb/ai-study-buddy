# Resume Description: AI Study Buddy

Here are professional bullet points and summaries you can add to your Resume/LinkedIn profile for this project.

---

## Project Title
**AI Study Buddy — Full Stack Web Application**

## Short Project Summary
*An AI-powered web platform designed to automate academic prep by generating practice assessments, context-aware lecture summaries, and customized daily study plans from user notes using Google Gemini API.*

---

## Detailed Bullet Points for Resume

- **Full-Stack Development:** Engineered a production-ready web application using **React.js (Vite)** on the frontend and **Java Spring Boot** on the backend, deploying to Vercel and Render respectively.
- **AI Core Integration:** Integrated **Google Gemini API** to generate context-aware multiple-choice quizzes (MCQs), markdown study summaries, and structured day-by-day learning timelines.
- **Robust Security Framework:** Designed and implemented a token-based authentication system using **Spring Security 6** and **JWT (JSON Web Tokens)**, securing REST endpoints and password hashing via BCrypt.
- **Relational Schema Design:** Structured database tables in **PostgreSQL** using Hibernate/JPA to manage users, study guides, serialized quiz histories, and progress statistics.
- **Premium User Interface:** Crafted a responsive dark-themed user interface utilizing **Tailwind CSS**, featuring glassmorphism elements, interactive progress dashboards, and mobile-friendly sidebars.
- **Analytics Visualization:** Programmed a dynamic, lightweight SVG-based area chart to visualize student quiz scores and progression without heavy external libraries.
- **Offline Resiliency:** Programmed a fallback mock API system inside the AI service class to ensure complete application functionality and local testing in the absence of external API keys.

---

## Core Technologies Profile
- **Frontend:** React.js, Tailwind CSS, Axios, React Router, Lucide Icons
- **Backend:** Java Spring Boot, Spring Data JPA, Spring Security, Maven, REST APIs
- **Database:** PostgreSQL (Neon), H2 Database (local development)
- **AI & Integrations:** Google Gemini API (gemini-2.5-flash), REST Clients
- **Tools/Deployment:** Docker, Git, Vercel, Render
