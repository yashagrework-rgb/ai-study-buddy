Machine Learning Algorithms & Concepts
========================================

1. INTRODUCTION TO MACHINE LEARNING
----------------------------------------
Machine Learning (ML) is a subset of AI that allows software applications to learn from data and improve accuracy over time without being explicitly programmed.
- Supervised Learning: Model learns from labeled training data (inputs mapped to known targets).
- Unsupervised Learning: Model finds patterns in unlabeled data (no target outcomes).
- Reinforcement Learning: Agent learns by interacting with an environment, maximizing cumulative reward via trial and error.

2. SUPERVISED LEARNING ALGORITHMS
----------------------------------------
- Regression (Continuous Target):
  * Linear Regression: Models relationship between independent variable(s) and a dependent variable using a linear line: y = mx + c. Uses Ordinary Least Squares (OLS) to minimize Mean Squared Error (MSE).
- Classification (Discrete Target):
  * Logistic Regression: Predicts probability of a binary outcome. Uses the Sigmoid function: g(z) = 1 / (1 + e^-z) to map output between 0 and 1.
  * Decision Trees: Flowchart-like tree structure where internal nodes test features, branches represent outcomes, and leaf nodes represent target decisions. Splits are based on Entropy, Information Gain, or Gini Impurity.
  * Random Forests: Ensemble method combining multiple Decision Trees using bagging (bootstrap aggregation) to reduce variance and overfitting.
  * Support Vector Machines (SVM): Finds the optimal hyperplane that maximizes the margin between different classes. Uses the Kernel Trick to map data into higher dimensions.
  * Naive Bayes: Probabilistic classifier based on Bayes' Theorem, assuming strong (naive) independence between features.
  * k-Nearest Neighbors (kNN): Instance-based learner. Classifies objects based on majority vote of their 'k' closest neighbors in feature space (e.g., using Euclidean distance).

3. UNSUPERVISED LEARNING ALGORITHMS
----------------------------------------
- Clustering:
  * k-Means Clustering: Partitions data into 'k' clusters. Iteratively assigns data points to the nearest cluster centroid, then updates centroids to the mean of assigned points. Matches elbow method to determine optimal k.
  * Hierarchical Clustering: Creates a tree-like hierarchy of clusters. Agglomerative (bottom-up) or Divisive (top-down).
- Dimensionality Reduction:
  * Principal Component Analysis (PCA): Linear transformation that project data into a lower-dimensional space, maximizing variance along new axes called Principal Components. Removes multicollinearity.

4. MODEL EVALUATION AND PERFORMANCE METRICS
----------------------------------------
- Confusion Matrix: Grid mapping Actual vs. Predicted classes.
- Metrics:
  * Accuracy = (TP + TN) / Total
  * Precision = TP / (TP + FP) (Focus on minimizing false positives)
  * Recall / Sensitivity = TP / (TP + FN) (Focus on minimizing false negatives)
  * F1-Score = Harmonic mean of Precision and Recall = 2 * (P * R) / (P + R)
  * ROC-AUC: Plot of True Positive Rate against False Positive Rate. AUC measures overall classification ability.
  * Regression Metrics: MSE, RMSE (Root Mean Squared Error), MAE (Mean Absolute Error), R-squared (coefficient of determination).

5. GENERALIZATION: UNDERFITTING & OVERFITTING
----------------------------------------
- Bias-Variance Tradeoff:
  * High Bias (Underfitting): Model is too simple, fails to capture data patterns. High error on both training and test data.
  * High Variance (Overfitting): Model is too complex, learns training data noise. Low error on training data but high error on test data.
- Regularization (Prevents Overfitting):
  * L1 Regularization (Lasso): Adds absolute value of coefficients as penalty. Can shrink coefficients to zero (performs feature selection).
  * L2 Regularization (Ridge): Adds squared magnitude of coefficients as penalty.
